@TMPL(expl_header.txt)@ Unrestricted DNS lookups

About this test: In order to test how your ISP's DNS resolver, the component which translates a mnemonic name such as "@NETALYZR_DOMAIN@" into an Internet address, the applet needs to be able to perform DNS queries.

Java does not always allow arbitrary lookups of names, depending on the security settings. This test determines whether the applet can look up a name which does not resolve to this server's address. If successful, this means the applet can perform all the DNS tests. @TMPL(expl_footer_ns_generic.txt)@
