@TMPL(expl_header.txt)@ DNS support for IPv6

About this test: In this test we measure whether your system supports looking up names that use IPv6 instead of, or in addition to, IPv4. This checks whether your system, your system's DNS server, and their interaction with Google will support IPv6.

What if this test reports a problem: If this test reports a problem it is probably related to bad NXDOMAIN wildcarding, which means your DNS server turns failing DNS name lookups into alternative (typically advertisement-enhanced) "suggestions". These can cause problems for applications other than your web browser.
